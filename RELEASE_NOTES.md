# CS2 Raven Mod v2.0 - Release Notes

## 🎯 Yeni Özellikler / New Features

### ✨ Varsayılan Olarak Aktif / Enabled by Default
- Tüm özellikler (Aimbot, Triggerbot, ESP) exe açıldığında otomatik aktif
- All features (Aimbot, Triggerbot, ESP) automatically enabled on startup

### 🎮 Menü Kontrolü / Menu Control
- **DELETE** tuşu ile menü aç/kapat
- **DELETE** key to open/close menu

### 🐛 Düzeltmeler / Bug Fixes
- Bomba sayacı artık her roundda düzgün çalışıyor
- Bomb timer now works correctly every round

## 📥 Kurulum / Installation

### Türkçe
1. **cs2-external-esp.exe** dosyasını indirin
2. Dosyayı çalıştırın (Visual Studio gerekmez!)
3. CS2 oyununu açın
4. Oyununuzun **Tam Ekran Pencereli** modda olduğundan emin olun
5. **DELETE** tuşuna basarak menüyü açın

### English
1. Download **cs2-external-esp.exe**
2. Run the file (No Visual Studio required!)
3. Open CS2 game
4. Make sure your game is in **Borderless Windowed** mode
5. Press **DELETE** key to open menu

## 🎮 Kontroller / Controls

| Tuş / Key | Fonksiyon / Function |
|-----------|---------------------|
| DELETE | Menü Aç/Kapat / Open/Close Menu |
| Mouse 5 (Yan Tuş) | Aimbot |
| Mouse 4 (Yan Tuş) | Triggerbot |
| Mouse Wheel | Menüde Kaydır / Scroll in Menu |
| END | Programdan Çık / Exit Program |
